# 一个关于中国地理的网页
网页上的所有图片和内容全部来自中国地理官网。网页主要有三大功能部分组成：轮播图，3D卡片，深浅模式切换。利用html、css、JaveScript来实现
## 主要功能
* **轮播图:**
    * 自动播放，鼠标放上去时会停止播放。
    * 点击左右切换按钮时会进行左右切换轮播图。
    * 有移动端的适配
* **3D交互卡片:**
    * 当鼠标放上去时卡片会出现悬浮的效果
    * 当鼠标放上去时卡片会进行反转显示对图片的介绍
    * 当点击卡片时卡片上的风景图片会放大，整体的背景会变暗。
* **深浅切换:**
    * 当点击右上角的太阳图标时页面会变成暗色。
    * 点击月亮图标时页面会变成浅色
