# 一个关于中国地理的网页
网页上的所有图片和内容全部来自中国地理官网。网页主要有三大功能部分组成：轮播图，3D卡片，深浅模式切换。利用html、css、JaveScript来实现
## 主要功能
* **沉浸式登录入口**
   * 提供一个全屏磨砂玻璃质感的登陆界面。
   * 点击`开启路程`之后，登陆页面平滑淡出，主界面显现，同时开始自动播放背景音乐。
   * 输入框有漂浮的效果
* **轮播图:**
    * 自动播放，鼠标放上去时会停止播放。
    * 点击左右切换按钮时会进行左右切换轮播图。
    * 有移动端的适配
    * 图片底部配有半透明渐变遮罩，清晰展示标题与简介。
* **3D交互卡片:**
    * 当鼠标放上去时卡片会出现悬浮的效果
    * 当鼠标放上去时卡片会进行反转显示对图片的介绍
    * 当点击卡片时卡片上的风景图片会放大，整体的背景会变暗。
* **视听体验**
    * 右上角悬浮音乐按钮，播放时，图标旋转变为光盘样式，并伴随旋转动画。暂停时，图标静止变为音符样式。
    * 顶部导航栏采用背景模糊滤镜技术，在页面滚动时保持内容清晰可见有现代感。
* **深浅切换:**
    * 当点击右上角的太阳图标时页面会变成暗色。
    * 点击月亮图标时页面会变成浅色
* **移动端适配**
    * 针对屏幕宽度小于768px的设备进行了适配
    * 隐藏复杂的导航链接。
    * 调整轮播图高度与文字大小。
    * 优化卡片尺寸与内边距，确保手机端也能获得良好的浏览体验。
